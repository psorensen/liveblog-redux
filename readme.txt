Gutenberg blocks for liveblogging. Newest entries first.

== Description ==

Adds two blocks:

* **Liveblog** – Container that holds all entries. Use "Add new entry" at the top to add updates. Entries are ordered newest first (reverse chronological).
* **Liveblog Entry** – Single update: timestamp, author(s), and rich content (paragraphs, images, etc.). Supports pinning and optional Co-Authors Plus.

Requires the parent Liveblog plugin.

== Installation ==

1. Install and activate the parent Liveblog plugin.
2. Install and activate this plugin.
3. Add the Liveblog block to a post. Add entries with the button at the top.
